# 🎙️ Terra Parte Podcast

## 🧠 Sobre o Projeto
Este projeto apresenta a criação e organização de um podcast focado em temas relevantes e atuais.

O objetivo é explorar ideias, reflexões e discussões através de conteúdo em áudio.

## 🎯 Objetivo
- Desenvolver um projeto de podcast
- Organizar conteúdo de forma estruturada
- Utilizar ferramentas digitais para produção e divulgação

## 🚀 Conteúdo
- Episódios do podcast
- Roteiros
- Ideias e temas

## 🛠️ Ferramentas utilizadas
- GitHub
- Ferramentas de gravação e edição de áudio
- IA (se usou, coloca aqui)

## 📂 Estrutura do Projeto
- `/roteiros` → scripts dos episódios
- `/episodios` → links ou descrições
- `/ideias` → temas futuros
